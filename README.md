
# Om.term

Terminal utilities.


### NOTES

* To see what capabilities your terminal supports, run: `infocmp`  
  The capability names (the part before the =) can be looked up in the terminfo man page for a description of what they do.

* Use hexdump to check how it's encoded:  
  `echo -n ☠ | hexdump`
